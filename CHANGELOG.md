# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-06-30

### Added
- **AI-Powered Itinerary Suggestions**: Seeding of destination-curated sightseeing and dining activities.
- **Real-Time Boarding Pass System**: Secure QR code generation controlled via Driver Portal boarding window commands.
- **Group Chat**: Integrated Firebase realtime messaging module inside the package details page.
- **Driver Updates**: Direct announcement timeline for delays, alerts, and pickup notifications.
- **Trip Members List**: Roster showing passenger check-in states and driver verification badges.
- **Live Status Badges**: Progress indicator strip mapping current checkout and payment states.
- **Multi-Budget & Expense Sync**: Multiple budget limits management with automatic currency updates.
